## Stacks utilizadas

* .NET Core 7 
* Angular 16.1.1 `Dependências: Node 18.16.1 / Python 3.11.3`
* Microsoft SQL Server 2019

## Recursos utilizados
* Domain Driven Design
* Onion Architecture
* Injeção de dependência
* Entity framework
* Formulário reativo
* Validações em tempo real
